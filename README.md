# 🚑 Smart Accident Detection and Emergency Response System

## Overview

A prototype IoT-based accident detection system developed using Arduino Uno, GPS, GSM, and vibration sensing technologies. The project demonstrates how embedded systems can assist in improving road safety by detecting vehicle impacts and preparing emergency notifications.

> **Note:** This repository is intended for educational and demonstration purposes. The included Arduino sketch focuses on sensor monitoring, countdown logic, and system workflow. Communication features are intentionally left as placeholders.

## Features

- Vehicle impact detection using SW-420 vibration sensor
- GPS location acquisition (NEO-6M)
- 15-second confirmation countdown
- Cancel button for false alarms
- Buzzer and LED status indication
- Modular Arduino code

## Hardware

- Arduino Uno
- SIM900A GSM Module
- NEO-6M GPS Module
- SW-420 Vibration Sensor
- Buzzer
- LEDs
- Push Button
- LM2596 Buck Converter
- 12V Battery
